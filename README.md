## **⚒️ Teach Stack**
- Language: `Swift`
- IDE: `Xcode`
- Target iOS Version: iOS 16 and above
- Framework: `SwiftUI`, `UIKit`, `WebKit`, `MapKit`, `Combine`
- Architecture: `MVVM` (Model-View-ViewModel)
- Database: Google Firebase
